## What is Convert?

A **modern unit converter** built for the web. No bloat, no ads, no tracking.

Convert handles everything from **everyday conversions** like kilometers to miles, to **specialized tools** for developers, engineers, and scientists.

Works **offline** as a Progressive Web App - install it once and use it anywhere.

</td>

## Quick Start

</div>

<br/>

```bash
git clone https://github.com/pruthvi2805/Unit-Converter.git
cd Unit-Converter
npm install
npm run dev
```

<br/>

<table>
<tr>
<td>

| Command | Description |
|:--------|:------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |

</td>
<td width="50%">

```
convert/
├── src/
│   ├── components/   # UI components
│   ├── data/         # Converter definitions
│   ├── pages/        # Route pages
│   └── utils/        # Conversion logic
├── public/           # Static assets
└── scripts/          # Build scripts
```
